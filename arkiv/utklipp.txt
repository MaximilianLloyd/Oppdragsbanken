Jeg var litt for rask til å sende spørsmålet og glemte å legge ved en essensiell del. 
Jeg bruker nemlig masker generere SVG-ikonene mine, og ulike klasser for å styre farger og størrelse. 
Måten jeg setter det opp er ved å lage en maske, med og uten -webkit- for bredere kompatibilitet,  
som jeg deretter kan tilføre en farge og størrelse ved å kombinere det med en klasse som styrer dette. Her et eksempel på oppsettet:

CSS:
.maskeIndexOutline {
   -webkit-mask: url(/svg/im/im_home_outline.svg) no-repeat center;
   mask: url(/svg/im/im_home_outline.svg) no-repeat center;
}
.fargetFyll {
   background-color: var(--clr-tekst);
}
.ikonStr{
   width: 40px;
   height: 40px;
}
.kategoriKnappIkon {
   position: absolute;
   top: 50%;
   transform: translate(-10%, -50%);
   left: 0;
   border-radius: 50rem;
   background-color: var(--clr-ixDBoks);
   border: 2px solid var(--clr-ramme);
   width: 50px;
   height: 50px;
   z-index: 100;
   padding: 0.2rem;
}
.kategoriLenke {
   position: relative;
   z-index: 80;
   height: 2.5rem;
}

Når alle klassene er definert, kan jeg for eksempelt sette opp HTML-dokumentet slik:
<a href="" class="kategoriLenke">
   <div class="kategoriKnappIkon">
      <div class="ikonStr maskeIndexOutline fargetFyll></div>
   </div>
</a>


Er det en måte å generere et div-element før lenketeksten i hovedmenyen, som har klassen "kategoriKnappIkon", og et div-element som barn med fargetFyll ikonStr og "iconClass" fra siteHierarchy?
